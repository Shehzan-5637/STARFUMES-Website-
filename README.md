# 🌟 STARFUMES Wesbite 

A Python Tkinter-based desktop application simulating a real-time shopping experience for STARFUMES, a premium perfume brand.
A Python Tkinter-based desktop application simulating a luxury perfume shopping experience for **STARFUMES**, a premium perfume brand. This project provides an interactive shopping interface with account creation, product selection, smart cart management, and automated billing.

---

## 🚀 Features

- **Account Setup:** Enter your name and receive a unique 4-digit Customer ID.  
- **Perfume Gallery:** Browse STARFUMES collection with live pricing.  
- **Smart Cart:** Add or remove items with real-time quantity updates.  
- **Billing Engine:** Automatically calculates taxes:
  - Cash on Delivery (COD) adds **5% tax**  
  - Card Payment adds **2% tax**  
- **Order Confirmation:** Final “Thank You” screen with total payment.
